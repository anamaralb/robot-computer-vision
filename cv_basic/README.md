# Basic Computer Vision (ROS2)

Computer vision project using OpenCV in ROS2.

This project implements basic image processing operations using the TIAGO simulator.

## Features

- Color space conversion
- Image filtering
- Histogram analysis
- Frequency filtering

## Technologies

ROS2  
C++  
OpenCV  
Robotics simulation  

## Notes

This project is based on the template provided in the Computer Vision course at Universidad Rey Juan Carlos.

Original template code is licensed under CC-BY-SA 4.0.

This repository contains my own implementation developed on top of that template.
