# Computer Vision (ROS2)

This repository contains several computer vision projects developed during my Robotics Engineering degree.

All projects were implemented in C++ using ROS2, OpenCV and PCL.

## Projects

- Basic image processing
- Filtering and histogram operations
- Morphology, geometry and 3D projection
- Final project with point cloud and object detection

## Technologies

ROS2  
C++  
OpenCV  
PCL  
Computer Vision  
Robotics  

## Notes

Some parts of these projects are based on templates and example code provided during the Computer Vision course at Universidad Rey Juan Carlos.
