# Morphology, Geometry and 3D (ROS2)

Computer vision project using OpenCV and PCL in ROS2.

This project implements image processing, 3D projection and point cloud operations
using the TIAGO simulator.

## Features

- Morphological operations
- 3D to 2D projection
- 2D to 3D projection using depth image
- Point cloud processing
- Plane detection using RANSAC
- Camera intrinsic and extrinsic parameters

## Technologies

ROS2  
C++  
OpenCV  
PCL  
Robotics simulation
