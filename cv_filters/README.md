# Image Processing and Filtering (ROS2)

Computer vision project using OpenCV in ROS2.

This project focuses on image processing techniques such as histogram manipulation,
edge detection and line detection.

## Features

- Histogram operations
- Image thresholding
- Hough transform
- Edge detection
- Frequency filtering

## Technologies

ROS2  
C++  
OpenCV  
Robotics simulation  

## Notes

This project is based on the template provided in the Computer Vision course at Universidad Rey Juan Carlos.
