# Final Computer Vision Project (ROS2)

Final project combining image processing and point cloud processing in ROS2.

The program processes RGB and depth images from the simulator and generates
different representations of the environment.

## Features

- RGB image generation from point cloud
- Point cloud generation from RGB and depth
- Image processing operations
- Optical flow / pattern recognition (extra features)
- Object detection configuration

## Technologies

ROS2  
C++  
OpenCV  
PCL  
Robotics simulation
